# Proflight

Escribir un componente usando Reactt que muestre una lista de logos de ProFlight.
<br><br/>
Criterios de aceptacion:
1. El componente debe tener 2 botones.
2. Cuando selleccione el boton "Siguiente" se debe resaltar el siguiente logo.
3. Cuando el ultimo logo este seleccionado y se presione el boton "Siguiente" se debe resaltar el primer logo.
4. Cuando se seleccione el boton "Anterior" se debe resaltar el logo anterior.
5. Cuando el primer logo este seleccionado y se presione el boton "Anterior" se debe resaltar el ultimo logo.
